TkTon это кошелек для TON (The Open Network) блокчейна.

Основные возможности:

* Возможность подключить несколько разных кошельков одновременно
* Возможность проверить любой адрес в сети
* Возможность проверить транзакции любого адреса в сети
* Использует lite-client с Merkle proofs (без посредника в виде HTTP REST)
