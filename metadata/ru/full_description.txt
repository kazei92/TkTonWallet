TkTon это кошелек Android с открытым исходным кодом для TON (The Open Network) блокчейна.

Основные возможности:

* Возможность подключить несколько разных кошельков одновременно
* Возможность проверит любой адрес в сети
* Возможность проверит транзакции любого адреса в сети
* Material Design
* Использует lite-client с Merkle proofs (без http endpoint)
